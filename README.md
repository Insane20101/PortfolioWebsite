# 🌐 Personal Portfolio Website

This is a responsive personal portfolio website built using **HTML, CSS, and JavaScript**. It showcases projects, skills, and contact information in a clean and interactive UI.

## 🚀 Features

* Interactive hero section with typing animation
* Project showcase with tech stack and links
* Skills section with modern technologies
* Fully responsive design
* Contact form and social media links

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, animations, media queries)
* JavaScript
* Typed.js
* Font Awesome

## 🎨 CSS Concepts Used

* **Flexbox Layout** for responsive alignment and spacing across sections
* **CSS Variables (`:root`)** for consistent theming (e.g., primary color)
* **Positioning (relative & absolute)** to create layered UI elements and decorative icons
* **Animations & Keyframes** for effects like floating icons, scaling images, and smooth transitions
* **Hover Effects & Transitions** to enhance interactivity (buttons, project cards, icons)
* **Media Queries** to ensure responsiveness across mobile, tablet, and desktop devices
* **Z-index & Layering** for managing overlapping elements and visual depth
* **Box Shadows & Gradients** to create modern UI styling and emphasis
